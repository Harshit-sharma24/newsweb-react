# 📰 NewsWeb - React News 

A modern and responsive **News Web Application** built using **React.js**.
It allows users to explore the latest news across different categories like Technology, Sports, Business, and more.

---

## 🚀 Features

* 🗂 Category-based news (General, Technology, Sports, Business)
* 🔄 Real-time news fetching using API
* 🎯 Clean and modern UI
* 📱 Fully responsive design
* 🧭 Smart Navbar with active category highlight
* ⚡ Fast and smooth user experience

---

## 🛠 Tech Stack

* React.js
* JavaScript (ES6+)
* CSS
* News API

---

## 🔐 Security

* API key stored securely using `.env`
* Sensitive data is not exposed publicly

---


## 📂 Installation & Setup

```bash
git clone https://github.com/your-username/newsweb-react.git
cd newsweb-react
npm install
npm run dev
```
